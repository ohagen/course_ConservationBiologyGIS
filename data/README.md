# Data directory

This folder holds all spatial datasets used in the tutorials.

## Files

| File | Format | CRS | Description |
|------|--------|-----|-------------|
| `species_observations.csv` | CSV | WGS 84 (EPSG:4326) | Georeferenced species occurrence records (synthetic example data) |
| `study_area.gpkg` | GeoPackage | WGS 84 (EPSG:4326) | Study-area polygon – generated by `data/create_sample_data.R` |
| `land_cover.tif` | GeoTIFF | WGS 84 (EPSG:4326) | Synthetic land-cover raster (4 classes) – generated by `data/create_sample_data.R` |

## Generating the GeoPackage and GeoTIFF

Run the helper script once to create `study_area.gpkg` and `land_cover.tif`:

```r
source("data/create_sample_data.R")
```

The script requires the `sf` and `terra` packages.

## Column description – species_observations.csv

| Column | Type | Description |
|--------|------|-------------|
| `species` | character | Latin binomial |
| `longitude` | numeric | Decimal degrees (WGS 84) |
| `latitude` | numeric | Decimal degrees (WGS 84) |
| `date` | date (YYYY-MM-DD) | Date of observation |
| `observer` | character | Observer surname |
| `notes` | character | Field notes |

## Data sources

The datasets in this repository are **synthetic** and generated for teaching
purposes only. For real conservation analyses, consider:

- [GBIF](https://www.gbif.org) – global biodiversity occurrence data
- [iNaturalist](https://www.inaturalist.org) – community science observations
- [Copernicus Land Monitoring Service](https://land.copernicus.eu) – European land-cover maps
- [OpenStreetMap](https://www.openstreetmap.org) – vector base maps
